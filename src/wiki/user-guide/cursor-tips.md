# 操作的小技巧

本文汇总了使用 Cursor 的实用技巧和快捷操作，帮助你在日常开发中更高效地使用这个强大的 AI 编辑器。

## 快捷键精通

### 1. 核心功能快捷键

#### AI 功能
- `Tab` - 接受代码补全建议
- `Cmd/Ctrl + K` - AI 代码生成
- `Cmd/Ctrl + L` - 打开 AI 聊天
- `Cmd/Ctrl + I` - 打开 AI 编辑器
- `Cmd/Ctrl + Shift + A` - 激活 Agent 助手

#### 编辑器操作
- `Cmd/Ctrl + P` - 快速打开文件
- `Cmd/Ctrl + Shift + P` - 命令面板
- `Cmd/Ctrl + B` - 切换侧边栏
- `Cmd/Ctrl + J` - 切换终端
- `Cmd/Ctrl + \` - 分屏

### 2. 编辑增强快捷键

#### 多光标编辑
- `Cmd/Ctrl + D` - 选择下一个匹配项
- `Cmd/Ctrl + Shift + L` - 选择所有匹配项
- `Alt + Click` - 添加光标
- `Cmd/Ctrl + Alt + ↑/↓` - 向上/下添加光标

#### 代码导航
- `Cmd/Ctrl + T` - 转到符号
- `F12` - 转到定义
- `Alt + F12` - 预览定义
- `Shift + F12` - 查看引用

#### 代码重构
- `F2` - 重命名符号
- `Cmd/Ctrl + .` - 快速修复/重构
- `Alt + ↑/↓` - 移动行
- `Cmd/Ctrl + /` - 切换行注释
